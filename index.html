An Aspiring Frontend Developer

Kirstine Pineda | Frontend Developer

As a self-taught Frontend Developer, I was faced with a challenge: how to make myself consistently competitive in the job market. I have always been passionate about computers and technology, but I had never taken any formal training in web development.

When pandemic hits, I decided to look into taking some courses. When I first started learning how to code, it felt like a completely new language—like learning a whole new language all over again. But with dedication and hard work, I was able to learn how to use HTML, CSS and JavaScript by myself, and with the help of my mentor.

It was a difficult decision for me because I didn't have any prior experience in this field. It was scary at first—I left my comfort zone but it turned out to be one of the best decisions I've ever made!
